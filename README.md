# Anchor of Faith Logistics Website

This package is ready for deployment on Vercel.

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Create a GitHub repo.
2. Upload all files from this folder.
3. In Vercel, import the GitHub repo.
4. Deploy.

## Domain setup

If your domain is managed in Squarespace, add:

- A record: `@` → `76.76.21.21`
- CNAME: `www` → `cname.vercel-dns.com`
